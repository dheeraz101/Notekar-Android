## Notekar v4.0.7

Signed release - built automatically from the branch.

### Features & Enhancements
- **In-App Download & Installation**: Seamlessly download and install new updates directly within the app, complete with progress tracking and MD5 checksum integrity checks.
- **Stable & Beta Update Tracks**: Switch between stable releases and pre-release developer beta builds under the new track selector settings.
- **Offline Commits Cache**: Persistently stores commit logs offline using SharedPreferences, showing a floating warning SnackBar when loading offline.
- **Dynamic VirusTotal Security Feed**: Displays clean scan ratios, scan execution dates, and verification colors inside the security card.
- **iOS 26 Style Cupertino Indicators**: Swapped all CircularProgressIndicator instances with premium, high-fidelity CupertinoActivityIndicator views.
- **Enhanced Settings Search**: Settings search now indexes Update Track, VirusTotal Scan, and Offline Commits Cache options.
- **Refined Layouts & FAQ**: Aligned card containers to standard layouts and updated Help documentation with offline and security references.

### Security and Integrity
NoteKar binaries undergo automated compilation and scanning.
