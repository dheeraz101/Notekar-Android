# NoteKar 2.0.0

NoteKar 2.0.0 is the first GitHub release build, rolling the unreleased 1.0.0/1.1.0 work into one cleaner Android release.

## What's new

- Redesigned Settings with iOS-style grouped cards, dividers, status pills, selected-state glow, and a clearer order by user intent.
- Added accessibility and customization controls for haptics, reduced haptics, reduced motion, larger text, high contrast, compact history, confirm delete, minimal clock, button labels, large controls, and the last saved hint.
- Improved History with Today, Yesterday, and Earlier sections, compact rows, a Single icon filter, better swipe delete visuals, and optional delete confirmation.
- Redesigned the main toolbar with smoother tap feedback, clearer Single and Two-Way mode states, and IN/OUT chips.
- Added opt-in GitHub raw remote notices, notification tap routing, GitHub Releases update checks, and clear up-to-date status in Settings.
- Improved Android backup visibility, launcher icons, email and GitHub icons, tap-delay alignment, note popup delay handling, and compact What’s New/Changelog sheets.
- Includes the original launch features: offline moment storage, Single and Two-Way logging, notes, history filters, exports, backup import, themes, default startup mode, and tap delay control.

## APK files

- `notekar-2.0.0-universal.apk` works on all supported Android ABIs.
- `notekar-2.0.0-arm64-v8a.apk` is recommended for most modern Android phones.
- `notekar-2.0.0-armeabi-v7a.apk` is for older 32-bit ARM devices.
- `notekar-2.0.0-x86_64.apk` is for x86_64 devices and emulators.

SHA-256 checksums are available in `SHA256SUMS.txt`.
