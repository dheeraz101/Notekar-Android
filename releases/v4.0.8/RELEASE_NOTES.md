## Notekar v4.0.8 [Beta]

Signed release - built automatically from the branch.

### Features & Enhancements
- **Windows-Style Update Classification System**: Releases are divided into Feature, Security, and Beta updates, with custom layouts and localized descriptions.
- **Dynamic Build Channel Badges**: Displays the local build channel track status (Stable Build, Beta Build, or Security Build) directly on the up-to-date panel.
- **Dynamic Suffix Edit Icon**: Focus states in the reminder message composer toggle the pencil edit icon to a checkmark in accent color.
- **Switch Track Spinner Overlay**: Triggers a 3-second transparent dialog overlay with Cupertino Activity Indicators when switching release tracks.
- **Upgraded Version Manager Scripts**: Releases script supports automatic increments via `-stable`, `-beta`, or `-security` switches.
