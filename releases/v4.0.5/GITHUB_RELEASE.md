# NoteKar 4.0.5

NoteKar 4.0.5 is a quality-of-life and localization release introducing full support for English, Hindi, and Spanish, contrast-optimized theme accent colors, and a dedicated Recently Deleted (Trash Bin) view.

### Highlights

- **Dynamic Translations**: Dedicated Language settings selector supporting English, Hindi, and Spanish with l10n support.
- **Recently Deleted (Trash Bin)**: Centrally manage deleted moments with individual/bulk restore and a 30-day auto-purge policy.
- **Visual Polish**: Dynamic contrast-adjusted accent colors across all themes and snappy (180ms) transition animations.
- **Hardened Resets**: Purge preferences, active/trash databases, and locales during Factory Reset or Clear Data.
- **Aesthetic Refinements**: Minimal, neutral-colored update notices and localized search indexing.
