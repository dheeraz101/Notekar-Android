# NoteKar 4.0.1

Release date: June 1, 2026
Build: 10

## What's new

- Added optional home menu icon motion, disabled by default.
- Menu Icon Motion now respects Reduce Motion and turns off visually when Reduce Motion is enabled.
- Settings that depend on another setting now show a helpful notice instead of appearing active when unavailable.
- Highlight Seconds now depends on Show Seconds.
- Refined History and Settings bottom sheets, History delete feedback, and swipe-delete polish.
- Fixed App Lock immediate timing so Android screen-lock confirmation does not loop inside the app.
- Added clearer feedback while Android applies launcher icon changes.

## APKs

- `notekar-4.0.1-universal.apk` - works on most Android devices.
- `notekar-4.0.1-arm64-v8a.apk` - recommended for most modern phones.
- `notekar-4.0.1-armeabi-v7a.apk` - for older 32-bit phones.
- `notekar-4.0.1-x86_64.apk` - for x86_64 Android devices/emulators.

See `sha256.txt` for checksums.
