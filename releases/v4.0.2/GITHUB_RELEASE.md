# NoteKar 4.0.2

A small polish release focused on settings behavior, motion preferences, History sheet cleanup, and App Lock reliability.

## What's new

- Added optional home menu icon motion, disabled by default.
- Menu Icon Motion automatically respects Reduce Motion.
- Dependency-aware Settings now explain what must be enabled first, such as Show Seconds before Highlight Seconds.
- Improved History and Settings bottom-sheet polish.
- Refined History delete feedback and swipe-delete visuals.
- Fixed App Lock immediate timing so Android screen-lock confirmation does not loop.
- Added applying feedback for Android launcher icon changes.
