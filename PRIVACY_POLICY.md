# Privacy Policy: NoteKar Android

NoteKar Android is engineered with a strict **privacy-by-design and offline-first architecture**.

## Highlights

- 🔒 **100% Local Storage:** All timestamps, notes, and preferences stay on your device via Hive local database and `SharedPreferences`.
- 🌐 **No Cloud Data Sync:** NoteKar operates without user accounts or cloud database servers.
- 💾 **Android OS Auto-Backup Support:** Standard Android Google Drive system backup is supported for device transfer and data protection.
- 📡 **Minimal Permissions:**
  - `INTERNET`: Strictly used for software release checks (`health.json` / GitHub APIs) and bug announcements.
  - `POST_NOTIFICATIONS`: Used locally for update notices and optional user reminders.
- 🚫 **Zero Trackers / Ads:** No commercial ad networks, third-party analytics SDKs, or data tracking.
